Below are few ways to find out the AEM version.

### Using Touch UI:

- Go to the start page [http://localhost:4502/aem/start.html](http://localhost:4502/aem/start.html)
- Click on the Help button present on the top right side.
- Click on About Adobe Experience Manager
- You will be prompted with dialog which shows the AEM version.
  ![AEM version Check touch UI](/static/posts/images/aem-verion-check-touchui.png)

### Using Classic UI:

Go to Welcome Page [http://localhost:4502/libs/cq/core/content/welcome.html](http://localhost:4502/libs/cq/core/content/welcome.html)

![AEM version Check touch UI](/static/posts/images/aem-verion-check-classicui.png)

### Using System Console

Go to System Console [http://localhost:4502/system/console/status-productinfo](http://localhost:4502/system/console/status-productinfo)

![AEM version Check touch UI](/static/posts/images/aem-verion-check-system-console.png)
